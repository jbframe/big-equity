
Prerequisites:
1/ miniconda package manager is installed on an osx-arm64 or standard linux computer
2/ Download and unpack file named assignment_4_code.zip from this shared directory https://gatech.box.com/s/1r1vbr0odje0hwqtdcww7lpow2k8xput 


Steps to run:
1/ In a terminal navigate to the assignment_4_code directory
2/ Execute $ conda env create --name be -f be.yml
3/ Execute $ conda activate be
4/ Execute $ jupyter lab
5/ Within the web jupyter interface navigate to jframe9-code-assignment-4.ipynb and run all cells


Steps to update env:
1/ Execute $ conda env update --name be -f be.yml
